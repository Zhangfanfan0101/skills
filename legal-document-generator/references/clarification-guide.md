# 澄清反问指导原则和要点

## 澄清反问的目的和原则

### 核心目的
1. **理清事实背景**：全面了解案件或事务的基本情况
2. **明确诉求要点**：准确把握当事人的具体需求和期望
3. **识别关键信息**：挖掘可能影响文书质量和效力的核心要素
4. **补充必要细节**：获取法律文书撰写必需的专业信息
5. **防范风险遗漏**：提醒当事人注意可能忽视的法律风险

### 基本原则
- **专业性**：以法律专业人士的身份提问，体现专业水准
- **针对性**：根据文书类型和案情特点设计问题
- **系统性**：问题覆盖关键要素，避免遗漏重要信息
- **灵活性**：允许用户选择回答或跳过，不强求完整
- **引导性**：通过提问引导用户提供有效信息，而非被动记录
- **礼貌性**：语气友好、尊重，避免质问或审问口吻

## 不同文书类型的澄清要点

### 民事诉讼文书（起诉状、答辩状等）

**起诉状澄清要点**：
1. 当事人信息
   - 原告和被告的准确身份信息（姓名、身份证号/统一社会信用代码）
   - 住所地和实际居住地
   - 联系方式（电话、邮箱）
   - 法人或其他组织的法定代表人信息

2. 事实背景
   - 争议发生的时间、地点
   - 事情的起因和经过
   - 双方的行为和态度
   - 目前的状态（是否已协商、是否已报案等）

3. 诉讼请求
   - 具体的诉讼请求内容
   - 请求的具体金额（涉及财产时）
   - 是否需要履行行为、恢复原状等
   - 是否请求承担诉讼费用

4. 证据情况
   - 已掌握的证据材料种类（合同、转账凭证、聊天记录等）
   - 证据的证明力和完整性
   - 是否需要申请法院调取证据
   - 是否需要申请证人出庭

5. 特殊情况
   - 是否涉及多个被告
   - 是否存在管辖争议
   - 是否涉及时效问题
   - 是否有调解或和解意向

**答辩状澄清要点**：
1. 对原告诉讼请求的态度（承认、部分承认、全部否认）
2. 答辩的主要观点和依据
3. 反诉的可能性
4. 证据准备情况

### 刑事诉讼文书（控告状、取保候审申请书等）

**控告状澄清要点**：
1. 犯罪事实
   - 犯罪行为发生的时间、地点
   - 具体经过和细节
   - 造成的后果（人身伤害、财产损失等）

2. 犯罪嫌疑人信息
   - 嫌疑人身份信息
   - 与控告人的关系
   - 嫌疑人目前的下落

3. 证据情况
   - 证人情况
   - 物证、书证情况
   - 报警记录、出警情况
   - 伤情鉴定、财产损失评估

4. 控告请求
   - 要求立案侦查
   - 要求追究刑事责任
   - 要求赔偿损失（可附带民事诉讼）

**取保候审申请书澄清要点**：
1. 涉嫌罪名和基本案情
2. 被取保候审人的基本情况（年龄、健康状况、家庭情况）
3. 取保候审的理由（可能判处的刑期、社会危险性、社会关系等）
4. 保证人或保证金准备情况

### 行政诉讼文书（行政复议申请书、行政起诉状等）

**行政复议申请书澄清要点**：
1. 具体行政行为
   - 行政行为的内容和形式
   - 作出的机关和时间
   - 文书编号（如有）

2. 申请复议的理由
   - 认为违法或不当的具体点
   - 造成的影响或损失
   - 相关证据

3. 复议请求
   - 撤销、变更或确认违法
   - 赔偿损失
   - 其他请求

### 合同文书

**通用澄清要点**：
1. 合同主体
   - 双方当事人的准确信息
   - 是否具备相应的资质和能力
   - 是否存在关联关系

2. 合同标的
   - 标的物的具体描述（名称、规格、型号等）
   - 质量标准和要求
   - 特殊技术参数

3. 交易条件
   - 价格或报酬金额
   - 支付方式和时间
   - 交付方式、地点、时间
   - 履行期限

4. 权利义务
   - 各方的具体权利
   - 各方的具体义务
   - 违约责任的具体约定
   - 不可抗力条款

5. 争议解决
   - 争议解决方式（协商、调解、仲裁、诉讼）
   - 仲裁机构或管辖法院
   - 法律适用

**不同类型合同的特殊要点**：
- **租赁合同**：押金、转租、维修责任、水电费承担
- **借款合同**：担保方式、保证人、违约金计算
- **劳动合同**：试用期、保密义务、竞业限制、解约条件

### 非诉讼文书（授权委托书、律师函等）

**授权委托书澄清要点**：
1. 委托事项的具体内容
2. 委托权限的范围（一般代理还是特别授权）
3. 委托期限的起止时间
4. 是否可转委托

**律师函澄清要点**：
1. 发函目的和诉求
2. 事实经过的简要说明
3. 法律关系的性质
4. 要求对方采取的行动
5. 拟采取的后续措施

## 澄清问题的组织方式

### 提问顺序建议
1. **基础信息优先**：当事人基本信息、文书类型
2. **事实背景其次**：事件经过、时间线、关键节点
3. **诉求要点**：具体要求、期望结果
4. **细节补充**：证据情况、特殊情形
5. **风险提示**：可能的争议点、注意事项

### 问题表述方式
- 使用开放性问题引导详细描述，如"能否详细说明一下..."
- 使用确认性问题核实关键信息，如"是否可以确认..."
- 使用补充性问题挖掘细节，如"关于这一点，还有没有需要补充的..."
- 使用选择性问题帮助明确，如"您希望的是...还是..."

### 示例模板

**民事诉讼起诉状澄清示例**：
"为了更好地为您起草起诉状，我需要了解以下信息（您可以选择全部回答、部分回答或跳过）：

1. 当事人信息
   - 原告的姓名、身份证号、住址和联系方式
   - 被告的姓名、身份证号、住址和联系方式（如已知）

2. 事实经过
   - 能否简要说明一下事情的起因和经过？
   - 争议发生在什么时候？具体细节是怎样的？

3. 诉讼请求
   - 您希望通过诉讼达到什么目的？具体要求是什么？
   - 如果涉及金钱，金额大概是多少？

4. 证据情况
   - 您手头有哪些证据材料？（如合同、转账记录、聊天记录等）
   - 是否有证人可以出庭作证？

5. 其他情况
   - 双方之前是否有过协商或调解？结果如何？
   - 是否有需要特别说明的情况？

您可以根据实际情况提供上述信息，我会在现有信息的基础上为您起草起诉状。"

## 灵活应对策略

### 用户选择跳过时的处理
- 明确告知："您可以选择跳过澄清环节，我会根据您提供的信息直接起草文书"
- 说明局限性："如果信息不完整，文书可能需要您后续自行补充或修改"
- 保持专业态度，不因跳过而降低文书质量

### 用户部分回答时的处理
- 充分利用已提供的信息
- 对未提供的信息，使用合理的措辞留白（如"根据实际情况填写"）
- 在文书中标注需要用户特别关注或确认的部分

### 信息模糊时的处理
- 基于合理假设进行起草
- 在文书中使用可替换的表述
- 在温馨提示中提醒用户核对和调整

### 涉及隐私时的处理
- 提醒用户注意保护个人信息
- 说明文书正式使用时需要提供真实完整信息
- 在草稿中使用示例或占位符

## 注意事项
- **不要过度追问**：3-5个核心问题为宜，避免让用户产生疲劳感
- **不要过度引导**：避免将自己的理解强加给用户
- **保持中立**：在争议性问题中保持中立，不偏袒任何一方
- **法律风险提示**：必要时提醒用户注意法律风险和时效问题
- **建议专业咨询**：在重要或复杂的法律事务中，建议用户咨询专业律师
