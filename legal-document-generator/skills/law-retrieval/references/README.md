# 法律法规检索推荐 - 参考文档目录

本目录包含法律法规检索推荐相关的参考文档，帮助智能体更好地理解检索流程、评估标准和方法。

## 文档列表

### [authoritative-databases.md](authoritative-databases.md)
**权威法规数据库清单**

包含中国官方权威法律法规数据库的详细介绍：
- 国家法律法规数据库 (flk.npc.gov.cn)
- 国家行政法规数据库 (xzfg.moj.gov.cn)
- 国家规章库
- 国务院政策文件库
- 司法案例库
- 检索技巧和策略

**何时读取**:
- 需要确定法规查询的权威来源时
- 构建检索关键词时
- 验证法规来源的权威性时

### [quality-criteria.md](quality-criteria.md)
**法条推荐质量评估标准**

详细说明五维质量评估标准：
- 正确性：条文内容准确无误
- 时效性：法规现行有效
- 完整度：覆盖所需关键条款
- 相关度：与问题高度关联
- 有效性：明确可操作

**何时读取**:
- 筛选和评估检索到的法条时
- 验证推荐法条的质量时
- 解释推荐理由时

### [law-retrieval-process.md](law-retrieval-process.md)
**法条检索与校验标准流程（资深律师经验）**

资深律师的法条检索与校验标准流程，包含：
- 法律渊源与法条识别（人大立法、行政法规、司法解释、地方性法规）
- 方向与背景检索（搜索语法）
- 法律专业文章检索（《人民司法》、最高人民法院观点集成等）
- 传统法条检索方法（体系检索法、关键词检索法、案例追踪法）
- 法条的效力甄别（真实性、时效性、事项效力、时间效力审查）
- 交叉验证与校验

**何时读取**:
- 需要深入了解专业检索方法时
- 复杂案件需要多种检索方法时
- 需要验证法条效力时
- 需要进行交叉验证时

## 使用指南

### 读取顺序
1. 用户提出法律问题后，先不读取参考文档
2. 理解问题、归纳核心点后，根据需要选择性读取
3. 需要确定检索来源时 → 读取 `authoritative-databases.md`
4. 需要评估推荐质量时 → 读取 `quality-criteria.md`
5. 复杂检索需要专业方法时 → 读取 `law-retrieval-process.md`

### 读取策略
- **按需读取**: 只在必要时读取相关参考文档
- **精准定位**: 根据当前任务直接跳转到相关章节
- **避免过度读取**: 不要一次性读取所有参考文档

- 查找权威数据库 → `authoritative-databases.md#国家级数据库`
