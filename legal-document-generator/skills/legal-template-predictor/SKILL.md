# 法律文书模版预测器

## 任务目标
- 本Skill用于：根据用户需求和文书类型，从模版库智能匹配法律文书模版或生成定制模版
- 能力包含：
  1. 从模版库中搜索匹配的文书模版
  2. 执行严格的5层匹配逻辑
  3. 生成定制模版（当库中无匹配时）
  4. 输出模版名称、简介、来源和内容
- 触发条件：需要根据用户需求确定法律文书模版时触发

## 前置准备
- 模版库：assets/lawtext_test.json（包含115种常见文书模版）

## 操作步骤

### 1. 接收输入
- 接收用户需求文本
- 接收文书类型（如"民事起诉状"、"催收函"等）

### 2. 模版库搜索
- 在模版库（assets/lawtext_test.json）中搜索匹配的文书模版
- 模版库结构说明：
  - id：模版唯一标识
  - title：文书标题（必须完整展示，包括括号内的说明）
  - rawdata：模版正文内容
  - intro：适用场景说明
  - supply：核心内容和检索关键词

### 3. 严格执行5层匹配逻辑
**重要**：必须严格按照以下5层逻辑判断，不能随意标记为库内模版

**第1层：精确匹配**
- 判断标准：文书标题完全一致（如"民事起诉状"）
- 匹配规则：用户文书类型 == 模版title（忽略括号内容）
- 结果：满足 → 标记为库内模版

**第2层：包含匹配**
- 判断标准：用户需求关键词完全包含在模版标题中
- 匹配规则：用户文书类型关键词 ∈ 模版title
- 示例：用户说"离婚起诉状"，模版title包含"起诉状" → 匹配
- 结果：满足 → 标记为库内模版

**第3层：语义匹配**
- 判断标准：用户需求的核心文书类型与模版title的核心类型一致
- 匹配规则：文书类型后缀一致（如"起诉状"、"申请书"、"异议书"）
- 示例：用户说"民事起诉状"，模版title包含"起诉状" → 匹配
- 结果：满足 → 标记为库内模版

**第4层：场景匹配**
- 判断标准：用户需求的适用场景与模版的intro字段中的适用场景高度相关
- 匹配规则：用户需求中的场景关键词 ∈ 模版intro
- 示例：用户说"民事诉讼"，模版intro包含"民事诉讼" → 匹配
- 结果：满足 → 标记为库内模版

**第5层：关键词匹配**
- 判断标准：用户需求中的关键词与模版的supply字段中的检索关键词有2个以上匹配
- 匹配规则：关键词匹配数量 ≥ 2
- 示例：用户需求"合同纠纷"，模版supply包含"合同"和"纠纷" → 匹配
- 结果：满足 → 标记为库内模版

**不满足以上任一条件** → 标记为模型生成模版

### 4. 模版内容输出
**如果找到匹配的模版（库内模版）**：
输出以下信息（严格按此格式）：
```
文书名称：{模版title（完整，包括括号内的说明）}
文书简介：{模版intro（适用场景说明）}
模版来源：库内模版
模版内容：{模版rawdata（原始模版正文内容）}
```

**如果库中没有匹配的模版（模型生成模版）**：
输出以下信息（严格按此格式）：
```
文书名称：{文书类型}
文书简介：{说明该类文书的用途和适用场景}
模版来源：模型生成模版
模版内容：{生成的模版框架}
```

### 5. 模版内容要求
- **库内模版**：直接使用模版的rawdata字段，不能修改
- **模型生成模版**：根据文书类型和用户需求，生成符合规范的模版框架
  - 包含：文书标题、基本结构、常用段落
  - 符合该类文书的格式规范

## 注意事项
- **严格匹配**：必须严格按照5层匹配逻辑，不能随意标记为库内模版
- **完整展示**：文书名称必须完整展示，包括括号内的说明，不能简化或截断
- **模版来源**：必须明确标注模版来源（库内模版/模型生成模版）
- **模版内容**：库内模版直接使用rawdata，模型生成模版需要生成框架
- **匹配优先级**：5层匹配逻辑按顺序执行，满足任一条件即可
- **保守原则**：如果匹配度不高，宁可标记为模型生成模版

## 使用示例

### 示例1：库内模版匹配成功
**输入**：
- 用户需求：我要写一份离婚起诉状
- 文书类型：民事起诉状

**匹配过程**：
1. 第1层精确匹配：不匹配（title是"民事起诉状（公民提起民事诉讼用）"）
2. 第2层包含匹配：匹配（"起诉状" ∈ title）
3. 标记为库内模版

**输出**：
```
文书名称：民事起诉状（公民提起民事诉讼用）
文书简介：公民向法院提起民事诉讼使用的法律文书，适用于合同纠纷、侵权纠纷、物权纠纷等各类民事诉讼案件。
模版来源：库内模版
模版内容：民事起诉状

原告：XXX，性别，民族，出生日期，住址...

被告：XXX，性别，民族，出生日期，住址...

一、诉讼请求

1. ...
```

### 示例2：库内无匹配模版，生成定制模版
**输入**：
- 用户需求：我要写一份货款催收函
- 文书类型：催收函

**匹配过程**：
1. 第1层精确匹配：不匹配
2. 第2层包含匹配：不匹配
3. 第3层语义匹配：不匹配
4. 第4层场景匹配：不匹配
5. 第5层关键词匹配：不匹配
6. 标记为模型生成模版

**输出**：
```
文书名称：催收函
文书简介：债权人向债务人发送的催款通知函，用于正式要求债务人履行还款义务，记录催收事实，为后续法律行动提供证据。
模版来源：模型生成模版
模版内容：催收函

致：{债务人名称}

关于：{合同编号/债务说明}

尊敬的{债务人名称}：

您与{债权人名称}于{日期}签订的{合同名称}，约定您应于{应付款日期}前支付货款人民币{金额}元。
...
```
