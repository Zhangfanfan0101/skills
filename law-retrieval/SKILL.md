# 法律法规检索推荐

## 任务目标
- 本 Skill 用于:根据用户提出的法律问题，检索并推荐最相关、有效的法律法规条文
- 能力包含:问题理解与核心点归纳、检索方向制定、法规查询、五维质量评估、标准化输出
- 触发条件:用户明确要求检索**法律条文、法条、法规**（如"查一下xxx法条"、"根据民法典第x条"、"工伤保险条例第几条"等）
- **不触发条件**:用户要求检索**裁判案例、类案**（如"找下裁判案例"、"搜索类似案例"、"案例参考"等），此时应使用 legal-case-retrieval-v3 技能
- **输出要求**:仅输出法规条文内容，不输出任何分析、说明或建议

## 前置准备
- 网络访问:使用 web_search 工具访问权威法规数据库
- 无需额外依赖:所有功能由智能体和搜索工具完成

## 操作步骤
- 标准流程（用户可见步骤）:
  1. **问题理解与核心要点** (必须完整输出)
     稍等，我先分析您的问题，提取关键法律要点和检索方向。
     - 1.1 问题分析
       识别您提出的法律问题类型，分析涉及的法律关系、争议焦点和核心诉求。
     - 1.2 核心要点归纳
       提炼出需要解决的关键法律问题，归纳为2-5个核心要点。
     - 1.3 检索关键词提取
       确定核心法律术语、相关法规名称、案由类型等检索关键词。
  
  2. **检索方向与法规要点** (必须完整输出)
     稍等，我将先通过网络检索相关法律法规和检索策略。
     - 2.1 法规层级确定
       根据问题性质，确定需要检索的法规层级（法律、行政法规、部门规章、司法解释等）。
     - 2.2 制定机关识别
       确定可能制定相关法规的机关（全国人大、国务院、最高法、最高检、各部委等）。
     - 2.3 核心法规要点
       根据法律理论和实务经验，预判可能涉及的核心法规条款及其适用要点。
     - 2.4 关键词优化方向
       将围绕"核心法律术语+法规名称+具体情形"等关键词组合进行检索。
  
  3. **法规查询与信息收集** (必须完整输出)
     正在进行法规检索，稍候片刻。
     - 3.1 数据库访问
       优先访问 [references/authoritative-databases.md](references/authoritative-databases.md) 中列出的权威数据库。
     - 3.2 检索执行
       使用优化的关键词组合调用 web_search 工具进行法规检索。
     - 3.3 信息收集
       收集检索到的相关法规的完整条文、生效时间、效力状态等信息。
     - 3.4 检索概况
       汇总检索到的法规数量、来源分布、覆盖范围等信息。
  
  4. **法条推荐** (仅输出法条内容，标准化文本格式)
     - 4.1 相关性排序
       根据法条与问题的相关度进行排序。
     - 4.2 标准化输出
       严格按照标准格式输出法条内容：
       - 法规名称作为标题（使用 ## Markdown二级标题）
       - 条文序号为第二行
       - 条文内容为分段内容

- 内部处理步骤（用户不可见）:
  - 5. **质量评估** (内部处理，不输出给用户)
     - 5.1 真实性与准确性验证
       参考 [references/law-retrieval-process.md](references/law-retrieval-process.md) 中的效力甄别方法，核对法条内容与官方发布是否一致，识别虚假文件。
     - 5.2 时效性检查
       确认法规为现行有效，注意生效和废止时间，区分已公布但未施行的法条。
     - 5.3 事项效力审查
       确认法条在事项效力上适用于本案，注意法律文件标题和条文对适用事项范围的界定。
     - 5.4 时间效力审查
       确认法条在时间效力上适用于本案，区分民商事实体立法、实体司解、程序立法的时间效力规则。
     - 5.5 相关度与有效性评估
       评估法条与用户问题的关联程度，筛选现行有效、相关性高、可操作性强的法条。

- 可选分支:
  - 当 **问题涉及多个法律领域**:分别检索并交叉验证
  - 当 **法条存在修订冲突**:注明最新版本和生效时间
  - 当 **案例复杂需补充材料**:建议查阅相关司法解释或指导性案例

- 步骤1（问题理解与核心要点）：**必须输出**，包含1.1问题分析、1.2核心要点归纳、1.3检索关键词提取三个小步骤的完整信息

- 步骤4（法条推荐）：仅输出标准化文本格式的法条，不输出任何分析或说明
- 步骤5（质量评估）：**内部处理，完全不可见**，用户看不到任何相关信息

## 注意事项
- **场景区分**:本技能仅用于法律法规检索，如用户要求检索**裁判案例、类案分析**，请使用 **legal-case-retrieval-v3** 技能，不要使用本技能
- **步骤输出强制要求**:必须严格按照用户可见四个步骤的顺序依次输出每个步骤的完整信息，**禁止跳过任何步骤或合并步骤**
- **步骤完整性要求**:每个大步骤下的所有小步骤（如1.1、1.2、1.3等）都必须输出，不能省略任何小步骤
- **内部处理步骤**:步骤5（质量评估）为内部处理步骤，**完全不可见**，不要在任何地方输出步骤5的相关信息
- **输出规则**:步骤1-3输出完整的处理信息，步骤4仅输出标准化文本格式法条
- **最终输出格式严格限定**:法规名称作为标题、条文序号为第二行、条文内容为分段内容，禁止任何分析、解释、建议、推荐理由或额外说明
- **检索方法参考**:复杂检索可参考 [references/law-retrieval-process.md](references/law-retrieval-process.md) 中的体系检索法、关键词检索法、案例追踪法等方法
- **效力甄别**:严格按照资深律师标准进行法条效力甄别，包括真实性、准确性、时效性、事项效力、时间效力的全面审查
- **交叉验证**:重要法条需通过多数据源交叉验证，参考 [references/law-retrieval-process.md](references/law-retrieval-process.md) 第八部分
- 优先使用权威数据库，避免来源不明的法规
- 确保引用的法条为现行有效，注意生效和废止时间
- 提供的法条必须与用户问题高度相关，避免无关推荐
- 保持客观中立，不提供法律意见，仅提供法规依据
- 当问题复杂超出检索能力时，建议用户咨询专业律师

## 使用示例

### 示例1:劳动合同解除问题
**用户问题**:公司以"严重违反规章制度"为由解除劳动合同，是否合法？应该怎么办？

**步骤1 问题理解与核心要点** (必须完整输出)
稍等，我先分析您的问题，提取关键法律要点和检索方向。
- 1.1 问题分析
  您的问题是关于用人单位以"严重违反规章制度"为由单方解除劳动合同的合法性问题，涉及劳动合同解除的法定条件和程序。
- 1.2 核心要点归纳
  用人单位单方解除劳动合同的法定情形；规章制度的有效性要求；解除程序的合法性。
- 1.3 检索关键词提取
  劳动合同解除、严重违反规章制度、劳动合同法第三十九条。

**步骤2 检索方向与法规要点** (必须完整输出)
稍等，我将先通过网络检索相关法律法规和检索策略。
- 2.1 法规层级确定
  主要检索法律层面的《劳动合同法》，以及相关的司法解释和部门规章。
- 2.2 制定机关识别
  全国人大常委会（劳动合同法）、最高人民法院（司法解释）、人力资源和社会保障部（部门规章）。
- 2.3 核心法规要点
  《劳动合同法》第三十九条：用人单位可以解除劳动合同的六种法定情形；《劳动合同法》第四条：规章制度的制定程序和公示要求；《劳动合同法》第四十三条：单方解除工会的监督程序。
- 2.4 关键词优化方向
  将围绕"劳动合同法 第三十九条 严重违反规章制度"、"劳动合同解除 程序要求"、"规章制度 公示 有效性"等关键词组合进行检索。

**步骤3 法规查询与信息收集** (必须完整输出)
正在进行法规检索，稍候片刻。
- 3.1 数据库访问
  优先访问国家法律法规数据库（flk.npc.gov.cn）和最高人民法院司法解释数据库。
- 3.2 检索执行
  使用优化的关键词组合进行多轮检索，覆盖不同法规层级。
- 3.3 信息收集
  收集到相关法规3部，包括《劳动合同法》及配套司法解释。
- 3.4 检索概况
  检索到相关法规3部，主要来源：全国人大法律法规数据库、最高人民法院司法解释库。

**步骤4 法条推荐** (仅输出标准化文本格式)

## 《中华人民共和国劳动合同法》

第三十九条
劳动者有下列情形之一的，用人单位可以解除劳动合同：(一)在试用期间被证明不符合录用条件的；(二)严重违反用人单位的规章制度的；(三)严重失职，营私舞弊，给用人单位造成重大损害的；(四)劳动者同时与其他用人单位建立劳动关系，对完成本单位的工作任务造成严重影响，或者经用人单位提出，拒不改正的；(五)因本法第二十六条第一款第一项规定的情形致使劳动合同无效的；(六)被依法追究刑事责任的。

第四条
用人单位应当依法建立和完善劳动规章制度，保障劳动者享有劳动权利、履行劳动义务。用人单位在制定、修改或者决定有关劳动报酬、工作时间、休息休假、劳动安全卫生、保险福利、职工培训、劳动纪律以及劳动定额管理等直接涉及劳动者切身利益的规章制度或者重大事项时，应当经职工代表大会或者全体职工讨论，提出方案和意见，与工会或者职工代表平等协商确定。在规章制度和重大事项决定实施过程中，工会或者职工认为不适当的，有权向用人单位提出，通过协商予以修改完善。用人单位应当将直接涉及劳动者切身利益的规章制度和重大事项决定公示，或者告知劳动者。

第四十三条
用人单位单方解除劳动合同，应当事先将理由通知工会。用人单位违反法律、行政法规规定或者劳动合同约定的，工会有权要求用人单位纠正。用人单位应当研究工会的意见，并将处理结果书面通知工会。

**步骤4 法条推荐** (仅输出标准化文本格式)

## 《中华人民共和国民法典》

第五百七十七条
当事人一方不履行合同义务或者履行合同义务不符合约定的，应当承担继续履行、采取补救措施或者赔偿损失等违约责任。

第五百八十五条
当事人可以约定一方违约时应当根据违约情况向对方支付一定数额的违约金，也可以约定因违约产生的损失赔偿额的计算方法。约定的违约金低于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以增加；约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少。

## 《商品房销售管理办法》

第三十条
房地产开发企业应当按照合同约定，将符合交付使用条件的商品房按期交付给买受人。未能按期交付的，房地产开发企业应当承担违约责任。
