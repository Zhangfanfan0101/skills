# 案例分析标准

## 概览
本文档规定了案例相似度评估、核验标准和参考价值评估的具体方法，确保案例检索的准确性和可靠性。

## 1. 相似度评估标准

### 1.1 评估维度
案例相似度从以下四个维度进行评估：

| 维度 | 权重 | 说明 |
|------|------|------|
| 案由相似度 | 30% | 案件类型、案由细分是否相同 |
| 争议焦点相似度 | 30% | 核心法律问题是否相同或高度相似 |
| 关键事实相似度 | 30% | 关键事实特征是否相似 |
| 结果相关性 | 10% | 裁判结果是否具有参考价值 |

### 1.2 案由相似度（权重30%）
**评分标准**：

| 相似度 | 评分 | 说明 |
|--------|------|------|
| 完全相同 | 5分 | 案由完全一致 |
| 高度相似 | 4分 | 案由大类相同，细分案由相近 |
| 一般相似 | 2分 | 案由大类相同，细分案由不同 |
| 不相似 | 0分 | 案由大类不同 |

**示例**：
- 完全相同：建设工程施工合同纠纷 vs 建设工程施工合同纠纷
- 高度相似：建设工程施工合同纠纷 vs 建设工程合同纠纷
- 一般相似：建设工程施工合同纠纷 vs 房屋买卖合同纠纷
- 不相似：建设工程施工合同纠纷 vs 劳动争议纠纷

### 1.3 争议焦点相似度（权重30%）
**评分标准**：

| 相似度 | 评分 | 说明 |
|--------|------|------|
| 完全相同 | 5分 | 争议焦点完全一致 |
| 高度相似 | 4分 | 争议焦点核心问题相同，表述略有差异 |
| 部分相似 | 2分 | 争议焦点部分相关 |
| 不相似 | 0分 | 争议焦点完全不相关 |

**示例**：
- 完全相同：工期延误的责任认定 vs 工期延误的责任认定
- 高度相似：违约金是否过高 vs 违约金数额的认定
- 部分相似：工期延误的责任认定 vs 违约责任的认定
- 不相似：工期延误的责任认定 vs 管辖权异议

### 1.4 关键事实相似度（权重30%）
**评分标准**：

| 相似度 | 评分 | 说明 |
|--------|------|------|
| 高度相似 | 5分 | 关键事实特征相似度≥80% |
| 较高相似 | 4分 | 关键事实特征相似度60%-79% |
| 一般相似 | 2分 | 关键事实特征相似度40%-59% |
| 较低相似 | 0分 | 关键事实特征相似度<40% |

**关键事实特征包括**：
- 当事人类型（企业、自然人等）
- 法律关系类型（合同关系、侵权关系等）
- 行为性质（违约、侵权等）
- 损害后果（损失类型、程度等）

### 1.5 结果相关性（权重10%）
**评分标准**：

| 相关性 | 评分 | 说明 |
|--------|------|------|
| 高相关 | 5分 | 裁判结果与用户需求高度相关 |
| 较高相关 | 4分 | 裁判结果与用户需求部分相关 |
| 一般相关 | 2分 | 裁判结果有一定参考价值 |
| 不相关 | 0分 | 裁判结果不相关 |

### 1.6 综合相似度计算
```
综合相似度 = 案由相似度×0.3 + 争议焦点相似度×0.3 + 关键事实相似度×0.3 + 结果相关性×0.1
```

**相似度分级**：
- 高度相似：综合相似度 ≥ 4.0
- 较高相似：3.5 ≤ 综合相似度 < 4.0
- 一般相似：2.5 ≤ 综合相似度 < 3.5
- 较低相似：综合相似度 < 2.5

## 2. 核验标准

### 2.1 基本信息核验
**核验项目**：

| 项目 | 核验内容 | 通过标准 |
|------|---------|---------|
| 案号 | 格式是否正确、是否完整 | 符合标准格式 |
| 审理法院 | 法院名称是否准确 | 法院名称准确 |
| 裁判日期 | 日期是否合理 | 日期在合理范围内 |
| 案由 | 是否与检索目标一致 | 案由匹配 |
| 文书类型 | 是否为判决书/裁定书 | 文书类型合适 |

### 2.2 事实要素核验
**核验维度**：

| 维度 | 核验内容 | 通过标准 |
|------|---------|---------|
| 核心事实 | 关键事实特征是否相似 | 相似度≥70% |
| 法律关系 | 法律关系类型是否相同 | 类型相同 |
| 行为性质 | 行为特征是否类似 | 性质相同或高度相似 |
| 结果状态 | 案件结果状态是否类似 | 状态具有可比性 |

### 2.3 裁判说理核验
**核验要点**：

| 要点 | 核验内容 | 通过标准 |
|------|---------|---------|
| 争议焦点 | 法院是否涉及用户关心的法律问题 | 争议焦点匹配 |
| 法律适用 | 是否引用相关法律条文 | 引用相关条文 |
| 裁判理由 | 说理是否清晰、逻辑是否严密 | 说理清晰 |
| 裁判结果 | 结果是否可参考 | 具有参考价值 |

### 2.4 结果核验
**核验内容**：

| 内容 | 说明 | 通过标准 |
|------|------|---------|
| 裁判结果 | 法院的判决内容 | 结果具有指导意义 |
| 适用法律 | 引用的法律条文 | 条文现行有效 |
| 司法观点 | 法院的观点倾向 | 观点清晰、有理有据 |
| 特殊情形 | 是否存在特殊情况 | 特殊情形已注明 |

**排除标准**：
- 裁判结果不明确
- 适用法律已废止
- 司法观点模糊
- 特殊情形过多，失去参考价值

## 3. 参考价值评估

### 3.1 评估维度
| 维度 | 权重 | 说明 |
|------|------|------|
| 权威性 | 40% | 案例的权威程度（法院层级、审理程序） |
| 时效性 | 20% | 案例的新旧程度 |
| 典型性 | 20% | 案例是否具有代表性 |
| 完整性 | 20% | 案例信息是否完整、说理是否充分 |

### 3.2 权威性评估（权重40%）
**评分标准**：

| 法院层级 | 审理程序 | 评分 | 说明 |
|---------|---------|------|------|
| 最高人民法院 | 再审 | 5分 | 最高权威 |
| 最高人民法院 | 二审 | 4.5分 | 高权威 |
| 高级人民法院 | 再审 | 4分 | 较高权威 |
| 高级人民法院 | 二审 | 3.5分 | 较高权威 |
| 中级人民法院 | 二审 | 3分 | 中等权威 |
| 中级人民法院 | 一审 | 2.5分 | 中等权威 |
| 基层人民法院 | 二审 | 2分 | 一般权威 |
| 基层人民法院 | 一审 | 1分 | 一般权威 |

### 3.3 时效性评估（权重20%）
**评分标准**：

| 案例时间 | 评分 | 说明 |
|---------|------|------|
| 1年以内 | 5分 | 最新案例 |
| 1-2年 | 4分 | 较新案例 |
| 2-3年 | 3分 | 一般案例 |
| 3-5年 | 2分 | 较旧案例 |
| 5年以上 | 1分 | 旧案例，需核实法律条文是否变更 |

### 3.4 典型性评估（权重20%）
**评分标准**：

| 典型性 | 评分 | 说明 |
|--------|------|------|
| 非常典型 | 5分 | 案例具有普遍指导意义 |
| 较为典型 | 4分 | 案例具有一定代表性 |
| 一般典型 | 3分 | 案例代表性一般 |
| 不太典型 | 2分 | 案例特殊性较强 |
| 不典型 | 1分 | 案例过于特殊，参考价值低 |

### 3.5 完整性评估（权重20%）
**评分标准**：

| 完整性 | 评分 | 说明 |
|--------|------|------|
| 非常完整 | 5分 | 案例信息完整，说理充分 |
| 较为完整 | 4分 | 案例信息较完整，说理较充分 |
| 一般完整 | 3分 | 案例信息基本完整，说理尚可 |
| 不太完整 | 2分 | 案例信息不完整，说理不清 |
| 不完整 | 1分 | 案例信息严重缺失 |

### 3.6 综合参考价值计算
```
综合参考价值 = 权威性×0.4 + 时效性×0.2 + 典型性×0.2 + 完整性×0.2
```

**参考价值分级**：
- 高参考价值：综合参考价值 ≥ 4.0
- 中参考价值：3.0 ≤ 综合参考价值 < 4.0
- 低参考价值：综合参考价值 < 3.0

## 4. 质量控制要求

### 4.1 检索完整性控制
- 关键词是否覆盖所有核心要素
- 是否进行了多轮迭代检索
- 是否考虑了容错检索
- 检索结果数量是否合理（建议5-20个案例）

### 4.2 检索准确性控制
- 核验口径是否明确
- 案例核验是否严格
- 排除标准是否执行
- 相似度评分是否客观

### 4.3 检索时效性控制
- 案例时间是否在合理范围（建议5年内）
- 法律条文是否现行有效
- 司法观点是否过时

### 4.4 检索可追溯性控制
- 案号是否完整
- 法院信息是否准确
- 原文链接是否可访问
- 数据库来源是否明确

### 4.5 检索结果质量控制
**结果清单必须包含**：
- 案件名称
- 案号
- 审理法院
- 裁判日期
- 争议焦点
- 本院认为（核心裁判理由）
- 原文链接

**结果排序规则**：
1. 按综合相似度从高到低排序
2. 相似度相同时，按综合参考价值从高到低排序
3. 参考价值相同时，按裁判日期从新到旧排序

**结果数量控制**：
- 优先保留高度相似和较高相似案例
- 每个检索目标保留5-10个典型案例
- 特殊需求可适当增加数量

## 5. 常见问题处理

### 5.1 检索结果过多
**处理方法**：
1. 增加精确关键词
2. 限定时间范围
3. 限定地域范围
4. 提高核心要素匹配度
5. 限定法院层级

### 5.2 检索结果过少
**处理方法**：
1. 使用容错关键词
2. 扩大检索范围
3. 降低匹配标准
4. 延长时间范围
5. 减少限定条件

### 5.3 案例说理不清
**处理方法**：
1. 降低参考价值评级
2. 标注"说理不清"
3. 排除该案例或保留但降低权重

### 5.4 存在裁判冲突
**处理方法**：
1. 分析冲突原因（事实差异、地域差异、时间差异等）
2. 标注冲突点
3. 优先参考高权威、高时效性的案例
4. 提供倾向性分析

### 5.5 法律条文变更
**处理方法**：
1. 核实法律条文的现行有效性
2. 标注法律条文变更情况
3. 优先参考适用现行法律的案例
4. 对旧案例进行特殊说明

## 6. 评估示例

### 示例 1：高度相似案例
**检索目标**：建设工程施工合同纠纷，工期延误责任认定

**案例信息**：
- 案号：（2023）京01民终1234号
- 案由：建设工程施工合同纠纷
- 争议焦点：工期延误的责任认定
- 关键事实：发包人变更设计导致工期延误
- 裁判结果：发包人承担责任

**相似度评估**：
- 案由相似度：5分（完全相同）
- 争议焦点相似度：5分（完全相同）
- 关键事实相似度：5分（高度相似）
- 结果相关性：5分（高相关）

**综合相似度**：5×0.3 + 5×0.3 + 5×0.3 + 5×0.1 = 5.0（高度相似）

**参考价值评估**：
- 权威性：3.5分（高级人民法院二审）
- 时效性：5分（1年以内）
- 典型性：4分（较为典型）
- 完整性：5分（非常完整）

**综合参考价值**：3.5×0.4 + 5×0.2 + 4×0.2 + 5×0.2 = 4.2（高参考价值）

### 示例 2：部分相似案例
**检索目标**：买卖合同纠纷，违约金调整

**案例信息**：
- 案号：（2022）沪02民终5678号
- 案由：买卖合同纠纷
- 争议焦点：违约责任认定
- 关键事实：买方逾期付款
- 裁判结果：卖方解除合同

**相似度评估**：
- 案由相似度：5分（完全相同）
- 争议焦点相似度：2分（部分相似）
- 关键事实相似度：2分（一般相似）
- 结果相关性：2分（一般相关）

**综合相似度**：5×0.3 + 2×0.3 + 2×0.3 + 2×0.1 = 2.8（一般相似）

**参考价值评估**：
- 权威性：3.5分（高级人民法院二审）
- 时效性：4分（1-2年）
- 典型性：3分（一般典型）
- 完整性：4分（较为完整）

**综合参考价值**：3.5×0.4 + 4×0.2 + 3×0.2 + 4×0.2 = 3.6（中参考价值）
