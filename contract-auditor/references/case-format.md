## 概览
本文档规定了类案检索中案例信息的标准格式和要素提取规范，确保案例信息的一致性和可比较性。

## 1. 标准案例结构

### 1.1 基本信息
| 项目 | 说明 | 示例 |
|------|------|------|
| 案号 | 案件的唯一标识符 | （2023）京01民终1234号 |
| 案由 | 案件类型 | 建设工程施工合同纠纷 |
| 审理法院 | 审理案件的法院名称 | 北京市第一中级人民法院 |
| 裁判日期 | 判决作出的日期 | 2023年6月15日 |
| 文书类型 | 文书种类 | 判决书 |
| 审理程序 | 审理阶段 | 二审 |
| 当事人 | 原告、被告的基本信息 | 原告：A公司 被告：B公司 |

### 1.2 争议焦点
案件的核心法律问题，通常包括：
- 法律关系的认定
- 权利义务的确定
- 违约责任的认定
- 损害赔偿的计算
- 程序性问题等

### 1.3 事实认定
法院认定的重要事实，包括：
- 合同关系是否存在
- 合同内容如何约定
- 当事人履约情况
- 违约行为是否存在
- 损害后果如何等

### 1.4 裁判理由
法院判决的法律依据和说理：
- 引用的法律条文
- 法律适用的分析
- 事实与法律的结合
- 裁判逻辑的阐述

### 1.5 裁判结果
- 主要判决内容
- 权利义务的确定
- 履行义务的方式和时间
- 诉讼费用的承担
- 其他判决事项

### 1.6 原文链接
案例来源的访问链接，确保可追溯验证。

## 3. 标注规范

### 3.1 案号格式
标准格式：（年份）法院代字 审级代字 序号号
- 示例：（2023）京01民终1234号

### 3.2 日期格式
统一使用：YYYY年MM月DD日
- 示例：2023年6月15日

### 3.3 金额格式
统一使用：阿拉伯数字 + 单位
- 示例：100万元、5.5元

### 3.4 当事人标注
- 原告：标注为"原告"或"上诉人"
- 被告：标注为"被告"或"被上诉人"
- 第三人：标注为"第三人"

### 3.5 引用标注
- 法律条文：使用标准格式，如《民法典》第五百七十七条
- 司法解释：如《最高人民法院关于适用<中华人民共和国民法典>合同编通则若干问题的解释》第六十五条
- 司法文件：如《建设工程质量管理条例》第十六条

## 4. 完整示例

### 示例案例：建设工程施工合同纠纷

**基本信息**
| 项目 | 内容 |
|------|------|
| 案号 | （2023）京01民终1234号 |
| 案由 | 建设工程施工合同纠纷 |
| 审理法院 | 北京市第一中级人民法院 |
| 裁判日期 | 2023年6月15日 |
| 文书类型 | 判决书 |
| 审理程序 | 二审 |
| 当事人 | 上诉人（原审被告）：B房地产开发公司<br>被上诉人（原审原告）：A建筑公司 |

**争议焦点**
1. 工期延误的责任主体是谁？
2. 违约金数额是否过高，应否调整？

**事实认定**
- 2019年3月，A公司与B公司签订《建设工程施工合同》，约定A公司承建B公司开发的某住宅项目。
- 合同约定工期为18个月，自2019年4月1日至2020年9月30日。
- 2019年10月，B公司变更设计方案，要求A公司修改施工图纸。
- 因设计变更，A公司停工30天，复工后工期顺延30天。
- 2020年10月20日，工程竣工，较合同约定工期延误20天。
- B公司以A公司工期违约为由，拒绝支付剩余工程款并要求支付违约金。

**裁判理由**
**关于工期延误的责任：**
根据《民法典》第五百九十条的规定，因当事人一方的违约行为，致使合同目的不能实现的，对方可以解除合同。本案中，工期延误20天的原因，有10天系因B公司变更设计方案导致，该责任应由B公司承担；另10天系因A公司施工组织不当导致，该责任应由A公司承担。

**关于违约金数额：**
根据《民法典》第五百八十五条第二款的规定，约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少。本案中，合同约定的违约金为每天5000元，总违约金为10万元。A公司因工期延误给B公司造成的实际损失经鉴定为3万元，约定的违约金过分高于造成的损失，本院酌情调整为3万元。

**裁判结果**
1. B公司支付A公司剩余工程款800万元，于判决生效后10日内履行；
2. A公司支付B公司工期延误违约金3万元，于判决生效后10日内履行；
3. 驳回B公司的其他诉讼请求。

### 示例案例：买卖合同纠纷

**基本信息**
| 项目 | 内容 |
|------|------|
| 案号 | （2023）沪02民终5678号 |
| 案由 | 买卖合同纠纷 |
| 审理法院 | 上海市第二中级人民法院 |
| 裁判日期 | 2023年8月10日 |
| 文书类型 | 判决书 |
| 审理程序 | 二审 |
| 当事人 | 上诉人（原审原告）：C科技公司<br>被上诉人（原审被告）：D商贸公司 |

**争议焦点**
1. D公司交付的产品是否符合质量标准？
2. C公司是否有权解除合同并要求赔偿损失？

**事实认定**
- 2022年5月，C公司与D公司签订《设备采购合同》，约定D公司向C公司供应某型号设备10台。
- 合同约定设备质量标准为国家标准GB/T xxxx-2020。
- 2022年6月，D公司交付设备10台。
- C公司验收发现，其中3台设备不符合国家标准，存在质量问题。
- C公司要求D公司退货并赔偿损失，D公司拒绝。
- 经第三方机构检测，确认3台设备确实不符合国家标准。

**裁判理由**
**关于产品质量问题：**
根据《民法典》第六百一十五条的规定，出卖人应当按照约定的质量要求交付标的物。出卖人提供有关标的物质量说明的，交付的标的物应当符合该说明的质量要求。本案中，合同约定设备质量标准为国家标准，D公司交付的3台设备经检测不符合国家标准，构成违约。

**关于合同解除和损失赔偿：**
根据《民法典》第五百六十三条的规定，当事人一方迟延履行债务或者有其他违约行为致使不能实现合同目的，当事人可以解除合同。本案中，D公司交付的设备存在质量问题，致使C公司无法实现合同目的，C公司有权解除合同。根据《民法典》第五百八十四条的规定，当事人一方不履行合同义务或者履行合同义务不符合约定，造成对方损失的，损失赔偿额应当相当于因违约所造成的损失。C公司因设备质量问题造成的直接损失经评估为50万元，D公司应当予以赔偿。

**裁判结果**
1. 解除C公司与D公司签订的《设备采购合同》；
2. D公司返还C公司货款150万元，于判决生效后10日内履行；
3. D公司赔偿C公司损失50万元，于判决生效后10日内履行；
4. 驳回C公司的其他诉讼请求。
